RISE Fatigue Design Tool is a set of free web applications for statistical evaluating fatigue tests and loads that is developed at RISE Research Institutes of Sweden, Mechanics research group. RISE can also assist in fatigue assessment of materials and components. Our expertise ranges from load analysis through execution of fatigue testing to evaluation of testing.

Shiny Apps in RISE Fatigue Design Tool

  • Wöhler Curve Estimation: Evaluate the Wöhler curve (or SN-curve) from fatigue life data with constant amplitude loads.

  • Spectrum Wöhler Curve Estimation: Evaluate the Wöhler curve (or SN-curve) from fatigue life testing with spectrum loads.

  • Endurance Limit Estimation: Evaluate the endurance limit form staircase tests.

  • Load Analysis for Fatigue: Evaluate rainflow cycles, level crossings and load spectrum from a load history.

About RISE Fatigue Design Tool

RISE Fatigue Design Tool is a research platform implementing statistical tools for evaluating fatigue tests and loads to be used in fatigue design and testing. The apps are implemented as Shiny apps that are powered by R (a language and environment for statistical computing and graphics).

RISE Fatigue Design Tool, including the Shiny apps and the R-code for the fatigue calculations, has been developed by Pär Johannesson and Thomas Svensson.

The source code is found on GitHub, repository pj3950/fdt. The code is open source under MIT Licence.
